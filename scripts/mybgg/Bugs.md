What I've added to mybgg:

- I've added preorders and wishlist items
  - These are highlighted in the general list as well as put in separate subcategories on cards
- New filters:
  - Publishers, Designers, Artists, Year, Status/Wishlist
  - Added search box to filters - similar to Algolia
- Updated Min age to a slider
- Sort by last modified
- Hover over images - covers on card as well as expansions, accessories, etc.
  - Expansions and accessories have Rating, Year, and Wishlist on them
- Separate out Promos either on title or Family = Promotional
- Added Accessories, Reimplemented By, Reimplements, and Contains to cards with links and hovers
- Added Wishlist Comments & Comments
- Added Hovers text and icons on cards:
  - Title = Version
  - Player Count = Player Count Details (Recommended, expansion, etc)
  - Weight = more specificity
  - Age = community recommended age
  - Year (new) = Owned Version Year
  - Publisher (new) = Owned Version Publisher
  - Rating = number of Ratings
  - Rank = Other Rankings (Family, Strategy, War, etc)
  - Plays = First Play/Last Play
  - Status (new) = Wishlist
- Added many of the fields to the search box (alternative titles, expansions, accessories, etc)
- Added a clear button to the search box
- Lots of scrubbing titles and making them more consistent, help with sorting, etc.
- Add a version of the title with no special characters to help with search (e.g. Orleans)
- Ability to add links between Expansions/Accessories and Games - if they don't exist in BGG
- Allow for multiple copies of games
- Get the most specific version of details - custom > version specific > general game details
  - Mainly for titles and cover art
- Added a Placeholder game to link Expansions, Promos and Accessories that don't link to any other games
- Support for linking custom Unpublished prototypes as expansions to other games
- Exclude Box of Promos (by Family) from linking as an Expansion (Advent calendars, etc)

- Networking fix to prevent 202's from getting cached
- Support BGG Bearer token
